## Authentication

All API endpoints require authentication. Refer to the [authentication documentation](/docs/core) for details on how to authenticate your requests.

## Base URL

The base URL for all API endpoints is:

```
https://your-kaneo-instance.com/api
```

Replace `your-kaneo-instance.com` with your actual Kaneo instance domain.
